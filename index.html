Hacknet Nodes

The Hacknet is a global, decentralized network of machines. It is used by hackers all around the world to anonymously share computing power and perform distributed cyberattacks without the fear of being traced.

Here, you can purchase a Hacknet Node, a specialized machine that can connect and contribute its resources to the Hacknet network. This allows you to take a small percentage of profits from hacks performed on the network. Essentially, you are renting out your Node's computing power.

Each Hacknet Node you purchase will passively earn you money. Each Hacknet Node can be upgraded in order to increase its computing power and thereby increase the profit you earn from it.

    NUKE.exe This virus is used to gain root access to a machine if enough ports are opened. BruteSSH.exe This program executes a brute force attack that opens SSH ports FTPCrack.exe This program cracks open FTP ports relaySMTP.exe This program opens SMTP ports by redirecting data HTTPWorm.exe This virus opens up HTTP ports SQLInject.exe This virus opens SQL ports DeepscanV1.exe This program allows you to use the scan-analyze command with a depth up to 5 DeepscanV2.exe This program allows you to use the scan-analyze command with a depth up to 10 ServerProfiler.exe This program is used to display hacking and Netscript-related information about servers AutoLink.exe This program allows you to directly connect to other servers through the 'scan-analyze' command
